# Machine Learning Fundamentals - NCKH Project

Ttổng hợp các thuật toán Machine Learning cơ bản được triển khai bằng Python (Jupyter Notebook)

##  Danh sách các thuật toán đã triển khai
Dự án bao gồm các thuật toán quan trọng trong Machine Learning:

* **Linear Regression:** Dự báo giá trị liên tục.
* **Logistic Regression:** Phân loại nhị phân.
* **K-Nearest Neighbors (KNN):** Phân loại dựa trên lân cận.
* **Support Vector Machine (SVM):** Phân loại với siêu phẳng tối ưu.
* **Naive Bayes:** Phân loại dựa trên xác suất Bayes.
* **K-Means Clustering:** Thuật toán phân cụm không giám sát.
